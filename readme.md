### ☕ مراحل ها 
- #### 💻 دانلود کردن
     ```
    >> git clone https://github.com/sonicdev123/Nitro-Maker/new/main
    >> pip install -r requirements.txt
    ```
- #### 🖥️ ران کردن بات
      1 - وارد پوشه config بشید و بر روی فایلی که هست کلیک کنید
      2 - یک بات وب هوک بسازید و لینکش را داخل فایل قرار دهید
      3 - بعد لینک عکس وب هوک رو میزارید و اسم وب هوک رو میزارید
      4 - در فایلی که گفته شد میتونید اینکه چند تا نیترو بده رو تنظیم کنید
      5 - ترمینال رو باز کنید و کلمه python main.py رو تایپ کنید و از بات لذت ببرید
